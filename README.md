## Acesse ou visualize

## Rode o projeto
1. Clone este repositório:
```sh
  $ git clone endereço-do-repositório
```

<br>

2. Execute os seguintes comandos no terminal do VS Code:
```sh
  $ cd pasta-clonada-do-github
  $ npm install react-scripts --save 
  $ npm start 
```
